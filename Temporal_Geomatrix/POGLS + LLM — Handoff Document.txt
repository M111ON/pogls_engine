**Adapter layer ✅ ทดสอบผ่าน**
```c
llm_adapter_init(&ctx)
llm_adapter_add(&ctx, url, model)   // register specialist
llm_dispatch_lane(&ctx, lane, prompt, ...)   // single
llm_dispatch_parallel(&ctx, ids, prompts, n, ...)  // parallel
```
- TailsDNA cache: FNV-1a fingerprint, 16 slots ring buffer
- lane % 3 routing: 0=General, 1=Coder, 2=Math(fallback)
- parallel = pthread per specialist, total = slowest worker

### แผน integrate เข้า POGLS38

**B — World State → Prompt Serializer**
```c
// delta/cell state → text query
// ต้องออกแบบว่า POGLS world แปลงเป็น prompt ยังไง
typedef struct {
    uint32_t snap_id;
    uint8_t  lane;
    char     query[256];  // extracted from world state
} POGLSQuery;
```

**C — Plug เข้า Wire Pipeline**
```c
// ใน l38_gpu_batch_feed() loop หลัง gate pass
if (gr == GATE_PASS) {
    POGLSQuery q = pogls_extract_query(h_hil[i], h_lane[i]);
    llm_dispatch_lane(&llm_ctx, q.lane, q.query, 100, out, sizeof(out));
    pogls_store_response(fed, snap_id, out);  // delta commit
}
```

**D — Persistent TailsDNA Cache**
```
ตอนนี้: ring buffer 16 slots ใน RAM (หายตอน restart)
ต้องการ: file-backed cache บน I:\llama\cache\
key: FNV-1a hash → filename
value: response text
```

**E — เพิ่ม Thai specialist (optional)**
```
Typhoon2-1.5B → port 8084
lane % 4 routing เพิ่ม case 3 = Thai
```
